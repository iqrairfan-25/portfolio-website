# React Portfolio Website

## Overview
A personal portfolio website built using **React.js and Vite**, showcasing my web development skills, projects, and experience. The application follows a component-based architecture and is optimized for performance and responsiveness.

## Tech Stack
- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
- Bootstrap

## Features
- Responsive and modern UI
- Reusable React components
- Clean and scalable folder structure
- Fast build and hot module replacement with Vite

## Getting Started

### Prerequisites
- Node.js (v16 or later)
- npm or yarn

Cmd:
